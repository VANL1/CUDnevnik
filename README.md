# Решись!

Ниже — инструкция, которая поможет тебе начать работу над кейсом и запустить проект на своём компьютере.
Пожалуйста, следуй шагам внимательно - это избавит тебя от ошибок и ускорит работу.

Во время изучения инструкций и выполнения заданий ты можешь столкнуться с неизвестными тебе терминами, технологиями и инструментами. В этом случае поступай как настоящий разработчик - ищи документацию, гугли и проси LLM объяснить тебе непонятные моменты =)

# Как начать работать над кейсом

1. Настрой свое **окружение для работы** - установи все необходимые инструменты и скачай код проекта.  
   Инструкция: https://github.com/MikD1/reshis-2025/blob/main/project.md

2. Начни решение с задач **ревью кода** — это поможет разобраться в проекте.  
   Инструкция: https://github.com/MikD1/reshis-2025/blob/main/codereview.md

3. После этого переходи к задачам **по реализации функциональности**.  
   Инструкция: https://github.com/MikD1/reshis-2025/blob/main/issues.md
